hexo的安装问题

sudo apt-get install nodejs(安装node.js创建)
sudo apt install nodejs-legacy
sudo apt install npm(安装npm工具)
sudo npm config set registry https://registry.npm.taobao.org（更换npm的镜像源）
sudo npm config list （查看配置检查镜像源是否更换成功）
sudo npm install n -g（安装更新版本的工具n;提示：n是node.js的版本更新工具）
sudo n stable（更新node.js版本）可以看到有 installed：版号，说明更新成功
sudo npm install -g hexo（安装hexo）
